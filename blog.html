What is the Document Object Model (DOM)?

DOM can be used to connect HTML to JavaScript. The design and text of html and css can be changed and created dynamically by using Dom.

How do you select an element from the DOM?

Different types of html elements can be changed through different types of selectors in dom. For example: getElementById(), querySelector(),.querySelectorAll() etc.

What is event delegation in the context of the DOM, and why is it useful?

Event delegation is a technique in the DOM (Document Object Model) where a single event listener is attached to a parent element to handle events triggered by its child elements.

How do you manipulate an element's attributes and styles using the DOM?

html and css styles and various types of text data can be manipulated through dom. A variety of methods are used in the dom to manipulate these various attributes. For example: getAttribute(), setAttribute(), removeAttribute() etc.
